## 🎯 Key Features

### ✅ What You Can Do:

1. **Take Upcoming Doses**
   - See doses scheduled for next 24 hours
   - Tap "Take" button to mark as taken
   - Instant feedback with success message

2. **Take Past Doses**
   - See missed/pending doses from last 24 hours
   - Mark them as taken even if they're late
   - Helps maintain accurate records

3. **Skip Doses**
   - Mark doses as skipped if not needed
   - Different from "missed" status
   - Useful for optional medications

4. **Change History**
   - Tap any dose in history to edit
   - Change between Taken/Missed/Skipped
   - Fix mistakes or update records

### 📈 Automatic Updates:

After any action:
- ✅ Today's progress updates
- ✅ Streak counter updates
- ✅ Adherence percentage recalculates
- ✅ Recent activity refreshes
- ✅ Dose moves to appropriate section

## 🎨 Visual Indicators

### Status Colors:
- 🟢 **Green** = Taken (success)
- 🔴 **Red** = Missed (danger)
- 🟡 **Yellow** = Skipped (warning)
- 🔵 **Blue** = Scheduled (upcoming)

### Icons:
- ✓ Checkmark = Taken
- ✗ X mark = Missed
- ⊖ Minus = Skipped
- ⏰ Clock = Scheduled

## 💡 Pro Tips

1. **Check Pending Section Daily**
   - Review past pending doses each morning
   - Mark any you forgot to log
   - Keeps your adherence accurate

2. **Use History to Fix Mistakes**
   - Accidentally marked as missed? Change it!
   - Tap the dose and select correct status
   - Your stats will update automatically

3. **Pull to Refresh**
   - Swipe down on home page to refresh
   - Ensures you see latest doses
   - Updates all statistics

4. **Take Action Immediately**
   - Mark doses right after taking them
   - More accurate than remembering later
   - Builds good tracking habits

## 🔄 Workflow Examples

### Morning Routine:
```
1. Open app → Home page
2. Check "Pending (Last 24h)" section
3. Mark any forgotten evening doses
4. Check "Upcoming" for today's schedule
5. Take morning dose → Tap "Take" button
6. See progress update automatically
```

### Reviewing History:
```
1. Go to History tab
2. Select date range (Today/Week/Month)
3. Review adherence percentage
4. Tap any incorrect dose
5. Change status in modal
6. Confirm with success message
```

### Skipping Optional Medication:
```
1. See dose in upcoming list
2. Decide to skip (e.g., vitamin on full stomach)
3. Tap "Skip" button
4. Dose marked as skipped (not missed)
5. Doesn't negatively impact streak
```

## ❓ FAQ

**Q: Can I mark a dose as taken after it's marked as missed?**
A: Yes! Go to History, tap the dose, and change status to "Taken"

**Q: What's the difference between Missed and Skipped?**
A: Missed = forgot to take, Skipped = intentionally didn't take

**Q: How far back can I mark past doses?**
A: Currently shows last 24 hours of pending doses

**Q: Do past doses affect my streak?**
A: Yes, marking past doses as taken can help maintain your streak

**Q: Can I undo an action?**
A: Yes, go to History and change the status back
